# BPSALGOAi
NSE F&O AI Trading - Android Application

## Overview

BPSALGOAi is an Android application for NSE Futures & Options AI Trading.

## Quick Start

### Building the App

```bash
./build-android.sh
```

This will generate:
- Debug APK: `app/build/outputs/apk/debug/app-debug.apk`
- Release APK: `app/build/outputs/apk/release/app-release.apk`
- Release AAB: `app/build/outputs/bundle/release/app-release.aab`

### Installation

For testing:
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## Documentation

## Requirements

- Android 7.0 (API 24) or higher
- Internet connection for trading operations

## Features

- NSE F&O Trading Interface
- AI-powered trading algorithms
- Real-time market data integration
